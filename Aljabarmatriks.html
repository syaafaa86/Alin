  1. Matriks adalah Susunan Bilangan yang ditulis dalam baris dan kolom yang dibatasi oleh tanda kurung. didalam matriks itu ada elemen yaitu kolom dan baris. kolom itu susunan angka atau bilangan yang tegak dalam matriks sedangkan baris itu susunan angka atau bilangan yang mendatar dalam matriks. ada juga yang namanya ordo,Ordo matriks merupakan banyaknya baris dan kolom yang terdapat pada suatu matriks. misal dalam suatu matriks kolom 3 dan baris 2, ordo nya itu 2x3

C. Operasi Perkalian matriks adalah operasi mengalikan matriks mengalikan setiap elemen baris pada matriks pertama dengan elemen kolom pada matriks kedua. syarat mengalikan matriks itu kolom dari matriks pertama harus sama dengan baris dari matriks kedua misal matriks A ordo nya 2x3 dan matriks B ordo 3x3

# contoh: (AXD)
A = np.array([[3,2,4],[1,3,2],[2,3,5]])
print(A)
print()
D = np.array([[1,1,4],[1,3,2],[2,1,2]])
print(D)

[[3 2 4]
 [1 3 2]
 [2 3 5]]

[[1 1 4]
 [1 3 2]
 [2 1 2]]

result = [[0, 0, 0],
          [0, 0, 0],
          [0, 0, 0]]

# Perkalian matriks secara manual
for i in range(len(A)):  # Loop baris matriks A
    for j in range(len(D[0])):  # Loop kolom matriks B
        for k in range(len(D)):  # Loop kolom matriks A/baris matriks B
            result[i][j] += A[i][k] * D[k][j]

# Menampilkan hasil
print("Hasil perkalian matriks A dan D:")
for row in result:
    print(row)

Hasil perkalian matriks A dan D:
[13, 13, 24]
[8, 12, 14]
[15, 16, 24]

result = [[0, 0, 0],
          [0, 0, 0],
          [0, 0, 0]]

# Perkalian matriks secara manual
for i in range(len(A)):  # Loop baris matriks A
    for j in range(len(B[0])):  # Loop kolom matriks B
        for k in range(len(B)):  # Loop kolom matriks A/baris matriks B
            result[i][j] += A[i][k] * B[k][j]

# Menampilkan hasil
print("Hasil perkalian matriks A dan B:")
for row in result:
    print(row)

Hasil perkalian matriks A dan B:
[9, 15, 22]
[6, 15, 14]
[10, 19, 25]
